# Usage

1. **Install sass**  
    + `sudo npm install -g sass` - global installation (which I use)  
    + `npm install sass` - local installation (creates folders in repo)  
2. **Redefine color variables.** There are color variables at the top of
   each `scss` file. By default values are taken from `pywal`. If you
   don't use it. You should pass your own color values.  
3. Use `ez-compile.sh` or directly `sass` to **compile** `scss` files to
   `css`
